Integração com sistemas do cliente

Canal pelo qual SMs, status e eventos são trocados entre a Trouw e os sistemas do cliente. A escolha do canal define os dados de credencial e o comportamento de sincronização.

💡
O canal define como as SMs chegam e como os status voltam. API REST é o padrão recomendado para operações >500 SMs/mês.
